# Avareno Closed Beta — Tester Guide

Danke, dass du Avareno testest. Avareno ist ein privates Gedächtnis für
das echte Leben: Produkte, Belege, Garantien, Dokumente, Reparaturen und
offene Punkte an einem ruhigen Ort.

## Wer diese Beta ist

Eingeladene Einzelpersonen/Haushalte. Kein öffentlicher Zugang. Die Beta
speichert echte Daten — behandle sie wie ein echtes Werkzeug, aber lege
noch keine unersetzlichen Originaldokumente ausschließlich hier ab.

## Was du testen solltest (Kernreise)

1. Registrieren, Onboarding durchlaufen
2. Erstes Objekt anlegen (mit Art, Seriennummer, Garantie)
3. Beleg (PDF/Bild) hochladen und wieder öffnen
4. Erinnerung/offenen Punkt anlegen und abschließen
5. App neu laden — alles noch da?
6. Objekt bearbeiten, suchen, im Dashboard „Braucht Aufmerksamkeit" prüfen
7. Datenexport unter Profil → Privatsphäre ausprobieren
8. Optional: Home Assistant verbinden (eigene Instanz, Long-Lived Token)

## Bekannte Grenzen (kein Bug-Report nötig)

- KI-Belegauslesen ist noch vorbereitet, nicht echt
- Foto-Erfassung: geplant
- Kontolöschung wird als Antrag erfasst, noch nicht ausgeführt
- Keine E-Mail-/Push-Erinnerungen
- Preise sind ein Beta-Entwurf; Zahlungen laufen im Testmodus

## Einen Fehler melden

Melde Probleme per E-Mail an die Support-Adresse aus der App
(Profil → Hilfe) oder direkt an das Team.

Ein guter Report enthält:

- Was du erreichen wolltest (1 Satz)
- Was stattdessen passiert ist
- Schritte zum Nachstellen (so genau wie möglich)
- Gerät + Browser + ungefähre Uhrzeit
- Screenshot, wenn möglich

**Bitte NICHT mitschicken:** echte Belege/Dokumente mit persönlichen
Daten, Passwörter, Access-Tokens, vollständige Seriennummern oder
Screenshots, auf denen private Dokumente lesbar sind. Ein Ausschnitt
oder eine Beschreibung reicht.

## Datensicherheit in der Beta

- Deine Daten liegen auf einem einzelnen Server mit manuellen Backups
- Du kannst jederzeit alles als ZIP exportieren (Profil → Privatsphäre)
- Provider-Tokens (z. B. Home Assistant) werden verschlüsselt gespeichert
  und nie an den Browser gegeben
- Es gibt kein Tracking und keine Werbung
